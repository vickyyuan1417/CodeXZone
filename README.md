# CodeXZone Chatbot

This project contains a simple Flask web application that uses OpenAI's Chat Completion API to answer tech or coding questions. The chatbot mimics a casual and funny writing style with short sentences and emojis.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Set the `OPENAI_API_KEY` environment variable with your OpenAI API key.

3. Run the app:
   ```bash
   python app.py
   ```

4. Open `http://localhost:5000` in your browser to chat with the bot.
